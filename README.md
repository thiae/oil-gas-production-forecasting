# Oil-Gas-Production-Forecasting

A data engineering and machine learning project that analyzes the US oil and gas products.

## PROJECT GOALS
- Build an automated data pipeline from the EIA's API
- Perform time series analysis and forecasting
- Create an interactive visualization
- Deploy predictive models

## TECH STACK
- Python (Pandas, scikit-learn, requests)
- PostgreSQL
- Flask API
- Plotly/Dash for Visualization
